## Data Pipeline

### Log Types Generated

| Log Type | Records | Data Points |
|----------|---------|------------|
| Authentication | 7,000+ | Login/logout times, MFA, geo-location, success/fail |
| File Access | 35,000+ | File name, sensitivity, action, bytes transferred |
| USB/Removable | 2,000+ | Device ID, duration, bytes copied |
| Email | 26,000+ | Recipients, external flag, attachments, subject |
| Network Traffic | 60,000+ | IPs, ports, protocol, geolocation, bytes |
| VPN Events | 3,500+ | Geo-location, upload/download bytes |
| Endpoint/EDR | 32,000+ | Process name, suspicious flag, CPU/memory |
